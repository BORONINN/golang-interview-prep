В функции `PrintFive` происходит `deadlock`.

```go
func PrintFive() {  
    ch := make(chan int)  

    go func() {  
        for i := 0; i < 5; i++ {  
            ch <- i  
        }  
    }()  

    for n := range ch {  
        fmt.Println(n)  
    }  
}  

```
Исправьте код, чтобы избежать блокировки.

```go
package main

func PrintFive() {

    return nil

}
```

### Решение

```go

package main  
  
import "fmt"  
  
func PrintFive() {  
    ch := make(chan int)  
  
    go func() {  
        for i := 0; i < 5; i++ {  
            ch <- i  
        }  
        close(ch)  
    }()  
  
    for n := range ch {  
        fmt.Println(n)  
    }  
}
```
