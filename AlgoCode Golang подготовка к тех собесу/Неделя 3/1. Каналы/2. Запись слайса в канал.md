Дан слайс целых чисел `values`. Нужно написать функцию `SliceToChannel`, которая преобразует `values` в небуферизированный канал для записи.

Функция должна возвращать канал, в который будут последовательно отправляться все элементы из исходного слайса, а так же **закрыть канал**, когда все значения будут записаны.

```go
package main

func SliceToChannel(values []int) <-chan int {

    return nil

}
```

---
### Решение

```go

package main  
  
func SliceToChannel(values []int) <-chan int {  
    ch := make(chan int)  
  
    go func() {  
        for _, value := range values {  
            ch <- value  
        }  
        close(ch)  
    }()  
  
    return ch  
}
```
