Дан канал целых чисел `` `<-chan int` ``. Нужно написать функцию, которая вычитывает данные из канала в слайс и возвращает его.

**Пример работы:**

ch := make(chan int, 3)
ch <- 1
ch <- 2
ch <- 3
close(ch)

result := ChannelToSlice(ch) 

Вывод: result == []int{1, 2, 3}

```go
package main

func ChannelToSlice(ch <-chan int) []int {

    return nil

}
```

func generateChan(channelLen int, values ...int) <-chan int {  
    ch := make(chan int, channelLen)  
  
    go func() {  
        for _, v := range values {

---

### Решение
```go

package main  
  
func ChannelToSlice(ch <-chan int) []int {  
    values := make([]int, 0)  
    for value := range ch {  
        values = append(values, value)  
    }  
  
    return values  
}
```
