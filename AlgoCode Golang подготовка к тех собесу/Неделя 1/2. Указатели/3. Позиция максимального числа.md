легко

Дан массив `numbers` адресов целых чисел. Нужно написать функцию `` `GetMaxIndex` ``, которая находит индекс самого большого числа в массиве. 

Если есть два подходящих числа - вернуть индекс первого.

В случае, когда нельзя определить индекс (например, пустой массив) - возвращать `0`.

Пример 1:

Ввод: 
a := 3
b := 9
c := 5
d := 9
numbers := []*int{&a, &b, &c, &d}
Вывод: 1

Пример 2:

Ввод: 
numbers := []*int{}
Вывод: 0

Пример 3:

Ввод: 
a := 3
numbers := []*int{&a}
Вывод: 0

### Решение

```go
package main  
  
func GetMaxIndex(values []*int) int {  
    var maxValue *int  
    maxIndex := 0  
  
    for index, value := range values {  
        if value == nil {  
            continue  
        }  
  
        if maxValue == nil || *maxValue < *value {  
            maxValue = value  
            maxIndex = index  
        }  
    }  
  
    return maxIndex  
}
```
