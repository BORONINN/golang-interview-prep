Напишите функцию `` `numberWithSeparators` ``, которая преобразует строку `s` с целым числом в строку с разделителями тысяч в виде точек. Используй `strings.Builder`.

**Пример:**

Ввод: s = "1234567"
Вывод: "1.234.567"

**Пример:**

Ввод: s = "-12"
Вывод: "-12"

## Решение

```go

package main  
  
import "strings"  
  
const groupSize = 3  
  
func numberWithSeparators(s string) string {  
    if strings.HasPrefix(s, "-") {  
        return "-" + numberWithSeparators(s[1:])  
    }  
  
    n := len(s)  
    if n <= groupSize {  
        return s  
    }  
  
    first := n % groupSize  
    if first == 0 {  
        first = groupSize  
    }  
  
    var b strings.Builder  
    b.WriteString(s[:first])  
    for i := first; i < n; i += groupSize {  
        b.WriteByte('.')  
        b.WriteString(s[i : i+groupSize])  
    }  
    return b.String()  
}  
```
Важно использовать `strings.Builder`, а не реализовывать всю логику самому.
