средне

Напиши функцию, которая всегда возвращает ошибку, при этом **не импортируя для этого ни один из пакетов**. Текст ошибки может быть любым.

```go
package main

func handle() error {

    //...

}
```
---
### Тесты

```go

package main


import "testing"
type TestCase struct {
    name     string
    check    func() bool
}

var testCases = []TestCase{
    {
        name: "Проверка интерфейса",
        check: func() bool {
            actual := handle()

            _, ok := actual.(error)

            return ok
        },
    },
    {
        name: "Проверка ошибки",
        check: func() bool {
            actual := handle()

            return len(actual.Error()) > 0
        },
    },
}

func TestSolution(t *testing.T) {
	for _, tc := range testCases {
		t.Run(tc.name, func(t *testing.T) {
			if !tc.check() {
				t.Errorf("тест не пройден")
			}
		})
	}
}
```
___
### Решение

```go

package main

type CustomError struct {
    message string
}

func (e *CustomError) Error() string {
    return e.message
}

func handle() error {
    return &CustomError{"произошла ошибка"}
}
```


Суть задачи: проверить, насколько кандидат знаком с реализацией ошибок в Go.
