# golang-interview-prep
Golang Interview Prep - AlgoCode Конспекты и задачи для подготовки к техническому собеседованию по Go. Материал разбит на 4 недели: от строк, указателей и слайсов до каналов, мьютексов, паттернов concurrency (fan-in/fan-out, worker pool, semaphore) и пакета sync. Каждая тема содержит теорию, примеры кода и практические задачи с тестами.
