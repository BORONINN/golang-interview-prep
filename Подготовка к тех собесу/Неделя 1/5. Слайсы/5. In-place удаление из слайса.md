Дан слайс целых чисел `values` и целое число `pos`. Необходимо написать функцию `removeElement`, которая удаляет элемент, стоящий на позиции `pos`. Функция должна модифицировать исходный слайс, сохраняя порядок оставшихся элементов.

Если `` `pos` `` выходит за границы слайса (отрицательный или ≥ длины), необходимо вернуть исходный слайс.

**Пример 1:**

Ввод: values = [1, 2, 3, 4, 5], pos = 1
Вывод: [1, 3, 4, 5]

**Пример 2:**

Ввод: values = [1, 2, 3, 4, 5], pos = 5
Вывод: [1, 2, 3, 4, 5]

## Решение

```go

package main

func removeElement(values []int, pos int) []int {
    if len(values) == 0 || pos < 0 || pos >= len(values) {
        return values
    }

    // сдвигаем элементы влево начиная с pos
    for i := range len(values) {
        if i > pos {
            values[i-1] = values[i]
        }
    }

    return values[:len(values)-1]
}
```
