## Проблема срезов и UTF-8

Строки в Go используют кодировку `` `UTF8` ``. Это значит, что фактически один символ может занимать от 1 до 4 байт. Но мы помним, что строка — это массив байт. Проблему можно легко увидеть на примере:

```go
str := "Hello мир"
fmt.Println(str[:9]) // Hello м�

```
Проблема тут в том, что кириллица в `` `UTF8` `` занимает 2 байта, из-за этого у нас разломался символ и вывелась не вся строка

```go
str := "Hello мир"
fmt.Println(len(str)) // 12

```
## Тип rune и корректная работа с символами

Для работы с символами в Go есть тип руна (`` `rune` ``). Они сами ведут учет, сколько байт занимает символ, и разработчику не нужно дополнительно это учитывать:

```go
str := "Hello мир"
runeStr := []rune(str)
fmt.Println(runeStr)       // [72 101 108 108 111 32 1084 1080 1088]
fmt.Println(len(str))      // 12
fmt.Println(len(runeStr))  // 9

```

## Изменение строки через массив рун

Также массив рун можно изменять (потому что это просто массив). При этом мы можем вставлять сразу готовый символ и даже смайлик:

```go
str := "Hello мир"
runeStr[5] = 128512
runeStr[6] = 'п'
fmt.Println(string(runeStr)) // Hello😀пир
```
